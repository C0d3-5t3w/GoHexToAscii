# GoHexToAscii

* Converts hex files in a selected folder and puts the plain text ascii files in a desired folder.

# How to:

* sudo git clone "https://github.com/C0d3-5t3w/GoHexToAscii.git"

* cd GoHexToAscii

* sudo make build

* sudo chmod +x GoHexToAscii

* sudo ./GoHexToAscii
